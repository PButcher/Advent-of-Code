# Advent of Code 2023

Language: TypeScript

## Install

```
yarn install
```

## Run HTTP Server

Solutions for each day are now running at [localhost:3000](http://localhost:3000)

```
yarn start
```

## Build TypeScript

Builds `src/` to `build/`

```
yarn build
```

## Develop

Rebuilds `src/` to `build/` on save

```
yarn dev
```
